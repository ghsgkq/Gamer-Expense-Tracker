# 🎮 게이머 가계부 (Gamer's Expense Tracker)

**게이머 가계부**는 Google과 Apple 양대 스토어의 결제 내역을 한곳에서 통합하여 분석하고 관리할 수 있도록 돕는 웹 애플리케이션입니다. 이제 흩어져 있던 지출 내역을 하나의 대시보드에서 시각적으로 확인하고 자신의 게임 소비 패턴을 손쉽게 파악할 수 있습니다.

## ✨ 주요 기능

### 📊 **통합 분석 대시보드 (핵심 기능)**

  - **하나의 페이지에서 모든 것을\!**: `Google Play(.json)`, `Apple Store(.html)`, 그리고 `아이시움 라운지(.html)` 결제 내역 파일을 한 페이지에 모두 업로드하여 모든 지출을 통합된 데이터로 분석할 수 있습니다.
  - **플랫폼별 독립 분석**: 상단 탭 전환을 통해 전체/구글/애플/아이시움의 결제 데이터를 탭 이동만으로 손쉽게 전환하며 분리 분석할 수 있습니다.

### 🧾 **상세 분석 기능 (공통)**

  - **💱 다중 통화 지원**:

      - 결제 내역에 포함된 원화(₩), 달러($), 엔(¥) 등 다양한 통화를 자동으로 감지합니다.
      - 감지된 통화 목록이 드롭다운 메뉴로 제공되며, 원하는 통화를 선택하면 모든 요약, 차트, 내역이 해당 통화 기준으로 필터링되어 표시됩니다.

  - **📝 사용자 정의 키워드 관리**:

      - '기타'로 분류된 항목을 원하는 앱으로 재분류할 수 있도록 사용자가 직접 키워드를 추가하고 관리할 수 있습니다.
      - 앱 이름과 관련 키워드를 등록하고, 개별 키워드 또는 앱 전체를 삭제할 수 있습니다.
      - 키워드 목록이 길어질 경우를 대비해 실시간 검색 기능도 제공합니다.
      - (참고: 이 기능은 페이지를 새로고침하면 초기화됩니다.)

  - **💰 전체 결제 내역 요약**: 선택된 통화를 기준으로 모든 스토어의 총결제액과 가장 많은 금액을 지출한 '톱 스펜딩 게임'을 한눈에 보여줍니다.

  - **📈 전체 통계 시각화**: 
    - **소비 추이 (꺾은선/막대)**: 선택된 통화를 기준으로 상위 게임들의 결제액 누적 추이(6개 월 단위 꺾은선) 및 연도별 결제 합계(누적 막대)를 표시합니다.
    - **🍩 소비 분포 (도넛 차트)**: 지출액 상위 6개 게임의 소비 점유 비중(%)을 보여주며, 가독성을 위해 나머지 군소 게임은 '기타'로 통합 집계하여 시각화합니다.

  - **📥 결제 내역 엑셀 내보내기**:
    - **전체 내역 추출**: 업로드한 모든 플랫폼의 결제 내역을 모아 엑셀 파일(`.xlsx`)로 내보냅니다.
    - **개별 게임 내역 추출**: 특정 게임 상세 내역 화면에서 현재 보고 있는 게임의 거래 내역만 골라 엑셀 파일로 저장합니다.
    - **통화별 총액 자동 집계**: 화폐 단위(₩, $, ¥ 등)가 섞여 있어도 왜곡이 없도록 시트 맨 하단에 통화별 합계 요약 행을 자동 추가하며, 텍스트가 잘리지 않도록 열 너비가 자동 최적화됩니다.

  - **🔍 게임별 상세 분석**:

      - 월별 결제액 요약 및 상세 내역을 아코디언 UI로 제공합니다.
      - 월별 지출액 변동 추이를 막대그래프로 시각화합니다.
      - 상품명 검색 및 최신순 정렬을 지원하는 전체 결제 내역 테이블을 제공합니다.

  - **⭐ 특정 게임 맞춤 분석 (트릭컬 리바이브)**: '트릭컬 리바이브'를 선택하고 통화가 원화(KRW)일 경우, 주요 과금 상품에 대한 별도 요약 및 전용 필터링 기능을 제공합니다.

### 📱 **모바일 환경 최적화**

  - **반응형 디자인**: 스마트폰과 같은 작은 화면에서도 모든 기능이 자연스럽게 보이고 사용하기 편하도록 디자인되었습니다.
  - **카드형 테이블**: 화면이 좁아지면 '월별 결제 내역'과 '전체 결제 내역'의 테이블이 가로 스크롤 없는 깔끔한 카드 형태로 자동 전환되어 가독성을 높였습니다.

## 📁 프로젝트 구조

유지보수와 확장성을 높이기 위해 파일을 기능별로 분리했습니다.

```
.
├── index.html              # 통합 분석 페이지 (메인 대시보드)
├── recap.html              # 트릭컬 연말결산 리캡 페이지
├── css/
│   └── style.css           # 모든 페이지에 적용되는 공용 스타일시트
├── js/
│   ├── appKeywords.js      # 게임 이름 분류를 위한 키워드 사전 데이터
│   ├── parsers.js          # Google/Apple/아이시움 데이터 파싱 및 병합 로직
│   ├── common.js           # 공통 업데이트 내역 모달 및 UI 로직
│   ├── main.js             # 메인 대시보드 UI 이벤트 처리 및 차트 렌더링
│   └── recap.js            # 연말결산 영수증 카드 생성 및 데이터 가공 로직
└── guide/
    ├── apple_guide.html    # Apple Store 구매 내역 다운로드 가이드 페이지
    ├── guide.html          # Google Play 구매 내역 다운로드 가이드 페이지
    └── icium_guide.html    # 아이시움 라운지 결제 내역 저장 가이드 페이지
```

## 🚀 사용 방법

1.  `index.html` 파일을 엽니다.
2.  상단 탭에서 원하는 모드(통합 분석 / Google Play / Apple Store)를 선택합니다.
3.  `Order History.json` (Google), `문제 신고.html` (Apple) 또는 아이시움 라운지 `HTML` 파일을 업로드합니다. (하나만 올려도 분석 가능)
4.  데이터가 분석되면 하단에서 차트와 상세 내역을 확인합니다.

-----

  - **Google 데이터**: [Google Takeout](https://takeout.google.com/) → 'Google Play 스토어' → `Order History.json` 다운로드
  - **Apple 데이터**: [Apple 문제 신고](https://reportaproblem.apple.com/) → 전체 내역이 보일 때까지 스크롤 → 페이지 저장 (HTML)
  - **아이시움 데이터**: 아이시움 라운지 로그인 → 결제 내역 페이지 → 페이지 저장 (HTML)

## 🛠️ 사용 기술

  - **HTML5 / CSS3** (반응형 그리드 및 모바일 친화형 카드 레이아웃)
  - **JavaScript (Vanilla ES6)** (클라이언트 사이드 SPA 및 데이터 파싱)
  - **Chart.js** (누적 추이 라인 차트, 연간 합계 스택 바 차트, 지출 분포 도넛 차트 시각화)
  - **SheetJS (xlsx)** (결제 내역 엑셀 내보내기 및 자동 열 너비 보정 지원)
  - **html2canvas** (트릭컬 연말결산 카드뉴스 영수증의 이미지 파일 저장 기능 지원)
