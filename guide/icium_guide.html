✨ 아이시움 라운지 구매 내역 저장 가이드

⚠️ 중요: 개인정보 보호 경고

저장하는 HTML 파일에는 계정 정보 및 결제 관련 민감한 데이터가 포함되어 있을 수 있습니다.

절대로 이 파일을 다른 사람에게 공유하거나 공개된 장소에 업로드하지 마세요.

이 분석기는 사용자의 컴퓨터에서만 파일을 처리하며, 어떠한 데이터도 외부 서버로 전송하지 않습니다.

결제 내역 분석을 위해 아이시움 라운지(웹 상점)에서 구매 내역 HTML 파일을 저장하는 방법을 안내합니다.

  1. 먼저 아이시움 라운지 사이트에 접속하여 게임 계정으로 로그인합니다.

  2. 로그인 후, 상단 메뉴에 '마이페이지' 버튼을 클릭하여 이동합니다.

  3. '구매 내역' 버튼을 클릭하여 구매 내역으로 이동합니다.

  4. 구매 내역 페이지에서 더 불러오기를 눌러, 모든 구매 내역이 불러올 수 있도록 합니다.

  5. 모든 내역이 화면에 보인다면, 마우스 오른쪽 버튼을 클릭하여 '다른 이름으로 저장'을 선택합니다.

    (단축키: `Ctrl + S` 또는 `Cmd + S`)

    파일 형식을 '웹페이지, 전체'로 설정하여 저장해 주세요.

  6. 저장된 .html 파일을 분석기의 아이시움 라운지 업로드 영역에 넣어주세요.

참고: 이 과정에서 저장한 파일은 브라우저 메모리 상에서만 처리되며 외부로 유출되지 않습니다.
